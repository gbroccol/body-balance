# WEB Service

## Описание
Модуль отвечает за отображение интерфейса приложения и управляет рендерингом страниц с помощью Thymeleaf.

## API
- GET `/` - Главная страница ✅
- GET `/login` - Страница входа ✅
- GET `/authenticated` - Временная тестовая страница (нужно удалить) ✅❗

## Технологии
- Thymeleaf
